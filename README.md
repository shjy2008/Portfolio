# Next.js + TypeScript

This repository is a Next.js application using TypeScript and the App Router (`src/app`).

## Development

- Run `npm install`
- Run `npm run dev`
- Run `npm run lint`

## Structure

- `src/app`: application routes and API route handlers
- `src/features`: route-level UI features
- `src/components`: reusable UI components
- `src/projects`: portfolio project modules
